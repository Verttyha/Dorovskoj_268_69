Rectangle findMaxIdenticalSubmatrix ( const std::vector< std::vector< int > > & matrix)

Находит наибольшую прямоугольную подматрицу, заполненную одинаковыми значениями.

Находит наибольшую по площади подматрицу с одинаковыми элементами.

Последовательно проверяет все возможные прямоугольные подматрицы, начиная от каждой позиции (i, j). Сравнивает каждый элемент с эталонным значением — значением в (i, j).
