SearchMatrixIdenticalElements Documentation

Эта программа предназначена для поиска максимальной подматрицы одинаковых элементов в известной матрице с данными параметрами размеров.

Наименование программы: SearchMatrixIdenticalElements. Программа написана на языке C++.

Заголовочные файлы

  • errors.h — перечисление ErrorType и класс Error для генерации сообщений об ошибках.
  • functions.h — структура Rectangle и объявления функций для работы с матрицей: чтение, парсинг, валидация, запись, поиск подматрицы.

Исходные файлы

  • main.cpp — точка входа в программу.
  • findMaxIdenticalSubmatrix.cpp — функция по поиску наибольшей подматрицы максимальных элементов.
  • isValidMatrix.cpp — функция для проверки, что матрица корректная, все строки имеют одинаковое количество столбцов.
  • parseMatrixFromLines.cpp — функция преобразования строк в матрицу.
  • readLinesFromFile.cpp — функция по считыванию строк из файла.
  • readMatrixFromFile.cpp — функция по считыванию файла и созданию на его основе матрицы.
  • writeMatrixToFile.cpp — функция по записи матрицы в файл.
