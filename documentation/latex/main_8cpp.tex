Точка входа в программу. 

Читает имена входного и выходного файлов, затем\+:
\begin{DoxyItemize}
\item считывает матрицу из входного файла;
\item ищет максимально большую одинаковую подматрицу;
\item записывает результат в выходной файл.
\end{DoxyItemize}
